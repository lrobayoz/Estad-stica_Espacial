Modelamiento del Índice de Privación Económica promedio en Chicago (2014) utilizando regresión espacial

DESCRIPCION DE LA REGRESION ESPACIAL Y EL INDICE DE PRIVACION (Puede ser como general)

Objetivo general: Modelar el Hardship Index a partir de un modelo de regresión espacial.

  • Objetivos especificos:

    1. Realizar un análisis exploratorio que permita evidenciar el comportamiento del Índice de Privación Económica promedio en Chicago para el año 2014.

    2. Desarrollar un modelo de regresión auxiliar para preseleccionar las variables explicativas, el cual se acompañe de la verificación de supuestos distribucionales y de varianza en los residuos.

    3. Seleccionar la matriz \(W\) más adecuada para la corrección de autocorrelación espacial y ajustar el modelo para el Índice de Privación Económica promedio en Chicago para el año 2014.

La base de datos trabajada corresponde a Chicago Health and Socio-Economics (Center for Spatial Data Science, University of Chicago 2017), la cual contiene indicadores de salud pública y socio-economicos para las 77 áreas comunitarias de Chicago. Originalmente esta base incluye 86 variables y 77 observaciones.

En este trabajo se excluyen las variables de salud pública junto, debido a que los datos que contienen son de 2012 y no tienen su actualización para el 2014, fecha de la que son extraidos los datos de los indicadores socio-economicos. De tal manera las variables son:

Matriz W e Indice de Moran I

Para encontrar la matriz de pesos espaciales \(W\), se hizo uso del indice de moran. De esta forma el procedimiento correspondia a:

  1. Subir el .shp de los datos y realizar las modificaciones necesarias.

    ComChicago = sf::read_sf("C:/Users/laura/Downloads/comarea/ComArea_ACS14_f.shp")
ComChicago = st_transform(ComChicago,CRS("+init=epsg:21897"))

ComChicago$HISaveP <- ComChicago$HISave/100

coords <- st_coordinates(st_centroid(ComChicago))
xy0 = coords
trinb=tri2nb(xy0)

  2. Calcular la matriz \(W\) por distintos criterios (Torre, Reina, Vecinos) para la variable de interés.
