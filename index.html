Modelamiento del Índice de Privación Económica promedio en Chicago (2014) utilizando regresión espacial

Objetivo general: Modelar el Hardship Index a partir de un modelo de regresión espacial.

  • Objetivos específicos:

    1. Realizar un análisis exploratorio que permita evidenciar el comportamiento del Indice de Privación Económica promedio en Chicago para el año 2014.

    2. Desarrollar un modelo de regresión auxiliar para preseleccionar las variables explicativas, el cual se acompañe de la verificación de supuestos distribucionales y de varianza en los residuos.

    3. Seleccionar la matriz \(W\) más adecuada para la corrección de auto-correlación espacial y ajustar el modelo para el Indice de Privación Económica promedio en Chicago para el año 2014.

La base de datos trabajada corresponde a Chicago Health and Socio-Economics (Center for Spatial Data Science, University of Chicago 2017), la cual contiene indicadores de salud pública y socio-económicos para las 77 áreas comunitarias de Chicago. Originalmente esta base incluye 86 variables y 77 observaciones.

En este trabajo se excluyen las variables de salud pública junto, debido a que los datos que contienen son de 2012 y no tienen su actualización para el 2014, fecha de la que son extraídos los datos de los indicadores socio-económicos. De tal manera las variables son:

Al querer trabajar con el Índice de Privación Promedio en Chicago como variable respuesta, debemos tener en cuenta su naturaleza, es decir, es un Índice que se encuentra entre 0 y 100. Por tanto creemos pertinente considerar y formular tanto un modelo normal como un modelo beta que tome en cuenta esta naturaleza.

Modelo Normal

Selección de variables

Para la selección de variables se hizo uso del stepwise forward y de la regularización Lasso. El primero de ellos parte del modelo nulo y construye el modelo final a partir de una métrica de validación; y el segundo incluye una personalización que fuerza que algunos coeficientes de los predictores tiendan a cero.

Stepwise forward:

#:: Modelo completo:
modcomp = lm(HISave ~. , data = datosmod1)

#:: Modelo con intercepto:
modnull = lm(HISave ~1 , data = datosmod1)

#:: Utilizando el criterio forward:
forwardmod = step(modnull, scope = list(lower = modnull, upper = modcomp), 
                  direction = "forward")
